# SST
## Crear app
  `pnpm create sst notes`
  ` cd notes`
  `pnpm install`

## Levantar api
  `pnpm sst dev`

## Eliminar plantillas
  `pnpm sst remove API`
  `rm -r stacks/MyStack.ts packages/core/src/todo.ts packages/core/src/event.ts packages/functions/src/lambda.ts packages/functions/src/todo.ts packages/functions/src/events`
